What is this?
This is the browser-based interface for LoR. The program is being run on your local machine, not a remote server. What is LoR?
LoR Genome is a user-friendly workbench for comparitive genome analysis. It is designed with a focus on ease of use, interactivity, and reproducibility. Work in self-contained sessions, where you can upload data, run analyses, and track everything in one place. When you're done, you can export your session. A detailed manifest is kept for you, so you can always see exactly what you did and reproduce it later.
